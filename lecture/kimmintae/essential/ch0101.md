# (웹앱) 프로그래밍 어떻게 배울 것인가

## 프로그래밍 학습 도전에 실패하는 이유

- 방대한 양의 사전지식을 학습해야한다

### 시도 가능한 학습전략

- 일단 만들어 보며 우당탕 학습하기
- 단계를 나눠서 정리된 순서대로 학습을 진행하기
- 하이브리드 - 두가지 모두 시도
  - 단계를 나눠서 차근차근 배우면 재미가 없어 오랜시간 동기부여가 어렵다
