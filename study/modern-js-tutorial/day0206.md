## [파트 1 6.4] 오래된 var

- var 변수의 스코프는 함수 스코프이거나 전역 스코프입니다
  - 블록 스코프가 없습니다
- var 변수는 선언 위치와 상관없이 함수 본문이 시작되는 지점에서 정의됩니다
  - 선언은 호이스팅 되지만 할당은 호이스팅 되지 않습니다
- 즉시 실행 함수 표현식
  - 함수를 괄호로 감싸면 자바스크립트가 함수 선언문이 아닌 표현식으로 인식하도록 속일 수 있습니다
  - 함수 표현식은 이름이 없어도 괜찮고, 즉시 호출도 가능합니다
